# Welcome to the Pret Technical Assessment

This repository contains technical assessments designed to evaluate your skills in either Kotlin or Node.js, depending on your eligibility. Each subdirectory corresponds to a specific test and includes its own `README.md` file with detailed instructions.

## How to Get Started

1. Determine your assigned test (Kotlin or Node.js or Both).
2. Navigate to the corresponding subdirectory.
3. Follow the instructions provided in the `README.md` file within the subdirectory.

Good luck!
